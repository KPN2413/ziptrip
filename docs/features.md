# Features and Functionalities

## Frontend Features

1. Todo list page
   - Displays all todos.
   - Shows total, active and completed todo counts.
   - Provides form to create and edit todos.

2. Todo details page
   - Reads todo id from the query parameter.
   - Displays title, description, priority, due date, status, created date and updated date.

3. Search and filter
   - Search by title or description.
   - Filter by all, active and completed.

4. Todo actions
   - Create todo.
   - Edit todo.
   - Toggle completed status.
   - Delete todo.
   - View single todo details.

## Backend Features

1. Express API server
2. CRUD APIs for todos
3. JSON file based storage
4. Basic input validation
5. Error handling for missing todos and invalid requests
